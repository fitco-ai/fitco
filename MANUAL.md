# Fitco 프로젝트 개발 환경 셋업 및 AI 어시스턴트 사용 가이드

## 1. 개요

본 문서는 코딩 경험이 적거나 프로젝트에 처음 합류한 팀원들이 구글 안티그래비티(Antigravity) 및 앤트로픽 클로드 코드(Claude Code)를 활용하여 프로젝트를 세팅하고 개발에 참여할 수 있도록 돕기 위한 메뉴얼입니다. macOS 환경을 기준으로 작성되었습니다.

## 2. 사전 작업 설치 가이드

개발 환경을 구성하기 위해서는 몇 가지 필수 도구들의 설치가 필요합니다. 터미널(Terminal) 앱을 열고 아래의 순서대로 진행해 주십시오.

### 2.1 Git 설치 확인
Git은 파일의 변경점과 버전을 관리하는 도구입니다. macOS에는 기본적으로 내장되어 있습니다. 터미널에 아래 명령어를 입력하여 설치를 확인합니다.
```bash
git --version
```
만약 설치되어 있지 않다면 'Command Line Tools' 설치 안내 팝업이 뜹니다. 안내에 따라 설치를 완료해 주시기 바랍니다.

### 2.2 Node.js 설치
Node.js는 자바스크립트 실행 환경입니다. 버전 관리를 쉽게 하기 위해 nvm(Node Version Manager)을 사용하여 설치합니다.
터미널에 다음 두 줄을 차례대로 복사하여 붙여넣고 엔터를 누릅니다.
```bash
curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.40.4/install.sh | bash
\. "$HOME/.nvm/nvm.sh"
```
그 다음, 구동에 필요한 Node.js 24 버전을 설치합니다.
```bash
nvm install 24
```

### 2.3 패키지 매니저(pnpm) 설치
본 프로젝트 패키지 관리를 위해 pnpm을 사용합니다. 터미널에 아래 명령어를 입력하여 설치합니다.
```bash
npm install -g pnpm
```

### 2.4 안티그래비티(Antigravity) 설치
안티그래비티는 코드 작성 및 분석을 도와주는 AI 통합 개발 환경입니다. 
- 다운로드 링크: https://antigravity.google/ 
- 접속 후 macOS 버전에 맞는 설치 파일을 다운로드하여 설치해 주십시오.

### 2.5 클로드 코드(Claude Code) 설치
클로드 코드는 터미널 환경에서 직접 파일 시스템에 접근하여 코드를 분석하거나 터미널 명령어를 실행해 주는 AI 어시스턴트입니다.
터미널에 다음 명령어를 입력하여 설치합니다.
```bash
curl -fsSL https://claude.ai/install.sh | bash
```

## 3. 안티그래비티를 통한 프로젝트 설정

필수 프로그램 설치가 끝났다면 본 프로젝트 코드를 컴퓨터에 가져와야 합니다. 

1. 설치한 안티그래비티 앱을 실행합니다.
2. 구글 계정을 사용하여 로그인합니다.
3. 초기 화면에서 'Clone Repository' 버튼을 클릭합니다.
4. Repository URL 입력란에 다음 주소를 입력합니다:
   `https://github.com/fitco-ai/fitco.git`
5. 코드를 저장할 로컬 폴더 경로를 지정합니다.
6. 클론이 완료되면 'Open'을 눌러 해당 프로젝트를 엽니다.
7. 안티그래비티가 해당 폴더에 접근하기 위해 권한을 요청하는 팝업이 나타나면 이를 허용해 줍니다. 

## 4. 안티그래비티와 클로드 코드를 통한 협업 가이드

이제 환경 구축이 완료되었습니다. 두 AI 도구를 목적에 맞게 분리하여 사용하면 효율적으로 작업할 수 있습니다.

### 4.1 안티그래비티 활용법 (에디터 및 코드 분석)
안티그래비티는 화면상에서 코드 전체를 한눈에 보며 시스템의 구조나 특정 코드 라인의 의미를 파악할 때 유용합니다.
- 코드 탐색: 모르는 코드가 있다면 안티그래비티의 채팅 창에 "이 프로젝트의 폴더 구조와 각 폴더의 역할에 대해 설명해줘"라고 질문합니다.
- 코드 수정 요청: 작업 위치의 파일을 띄운 상태에서 "이 페이지에 새로운 버튼 기능을 만들어줘"와 같이 구체적으로 요청하면 AI가 관련된 코드를 분석하여 기능 추가를 돕습니다.

### 4.2 클로드 코드 확장 프로그램(Extension) 연동 및 활용
클로드 코드는 터미널 명령어로 단독 실행할 수도 있지만, 안티그래비티 내부 확장 프로그램으로 설치하면 더욱 편리하게 사용할 수 있습니다.

안티그래비티 내 설치 방법:
1. 안티그래비티 좌측 메뉴의 'Extensions(확장 프로그램)' 아이콘을 클릭합니다. (또는 `Cmd + Shift + X` 입력)
2. 검색창에 'Claude Code'를 검색하고 'Install(설치)' 버튼을 눌러 적용합니다.
3. 설치 후 안티그래비티 내부 터미널(`Cmd + \``)을 열고 `claude` 명령어를 입력하여 안티그래비티 화면을 벗어나지 않고 백그라운드 작업을 곧바로 지시할 수 있습니다.

활용 예시 (내부 터미널에서 입력):
- "이 프로젝트의 의존성 패키지들을 알아서 설치해 줘"
- "로컬 개발 서버를 실행해 줘"
- "서버를 띄우다 에러가 났는데, 터미널 로그를 확인하고 해결 방법을 적용해 줘"

### 4.3 두 도구의 조합 (시너지 효과)
1. 클로드 코드로 프로젝트 환경 설정(패키지 설치, 로컬 서버 실행 등) 및 에러 로그 분석과 같은 백그라운드 작업을 전담하게 합니다.
2. 안티그래비티는 화면에 파일을 띄워두고, 코드 작성, 리팩토링 및 복잡한 비즈니스 로직(예: UI 구성이나 API 연동)을 화면 단위로 파악하고 구현하는 데 사용합니다.
3. 개발 중 원인을 알 수 없는 복잡한 이슈가 있을 경우, 두 AI 도구의 답변을 교차 검증하며 문제를 보완해 나가는 방식으로 활용할 수 있습니다.
